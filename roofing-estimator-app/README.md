# Roofing Measurements & Estimates App

Standalone app focused only on roofing measurements, estimates, and carrier comparison.

## Open It

Open `index.html` directly in your browser.

## Features

- Property intake:
  - Address, state, coordinates, roof type
  - Plan area, perimeter, pitch
- Estimate generation:
  - Effective squares + waste
  - Severity + damage type influence
  - Regional state multiplier
- Measurement quality:
  - Sanity checks and warning messages
  - Quality score (0-100 style)
- Carrier scope comparison:
  - Xactimate-style parsing
  - RCV / ACV / Depreciation detection
  - Line-math mismatch checks
  - Missing-scope flags
- Settlement projection:
  - Deductible
  - Recoverable/non-recoverable depreciation
  - Initial and projected final payment
  - Estimated out-of-pocket vs estimator midpoint
