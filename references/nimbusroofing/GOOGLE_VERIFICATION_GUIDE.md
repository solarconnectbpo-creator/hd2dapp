## 🔍 Step 1: Google Search Console Verification

### Option A: HTML Meta Tag (Recommended - Easiest)

1. Go to [Google Search Console](https://search.google.com/search-console)
2. Click "Add Property" and enter: `https://www.nimbusroofing.com`
3. Choose **"HTML tag"** verification method
4. Google will provide a meta tag like:
   ```html
   <meta name="google-site-verification" content="YOUR_VERIFICATION_CODE_HERE" />
   ```
5. **Send this meta tag to your developer** to add to the website's `<head>` section
6. Click "Verify" in Google Search Console

### Option B: HTML File Upload

1. Download the HTML verification file from Google Search Console
2. Upload it to your website's public folder
3. Verify it's accessible at: `https://www.nimbusroofing.com/google[code].html`
4. Click "Verify"

### Option C: Domain DNS (Advanced)

1. Add a TXT record to your GoDaddy DNS settings
2. Use the verification code Google provides
3. Wait for DNS propagation (up to 48 hours)
4. Click "Verify"
